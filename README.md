# Codebase Intelligence Engine

> Deterministic pipeline that transforms a codebase into structured knowledge and validated system understanding.

---

## 🔒 Core Principles (DO NOT BREAK)

1. **Deterministic first, AI second**
   - AI never replaces structure
   - AI only interprets already-extracted data

2. **Traceability over cleverness**
   - Every output must include:
     - evidence
     - confidence
     - status

3. **Local correctness before global reasoning**
   - If function summaries are weak → everything above is invalid

4. **Strict abstraction boundaries**
   - Function → behavior only
   - File → responsibility only
   - Module → role only
   - System → architecture only

5. **Caching is mandatory**
   - No repeated AI calls for identical inputs

6. **Incremental build only**
   - No skipping phases
   - No jumping ahead

---

## 🧠 Final System Vision (DO NOT FORGET)

Pipeline:

Scanner  
→ Structure Builder  
→ Function Summarizer  
→ File Aggregator  
→ Module Builder  
→ Reasoning Engine (iterative)  
→ Output Generator  

Key differentiator:

> Iterative reasoning with evidence + confidence propagation
